# 🤖 Multi-Agent AI Research System

An AI-powered Multi-Agent Research System that automates web research by coordinating specialized AI agents for search, web scraping, content analysis, report generation, and response evaluation.

---

## 🚀 Features

- 🔍 AI-powered Search Agent
- 🌐 Web Content Scraping
- 🧠 Intelligent Research & Analysis
- 📝 Structured Report Generation
- ✅ Feedback & Quality Evaluation
- 💬 Interactive Streamlit Interface
- 🔄 Modular Multi-Agent Architecture
- 🤖 Local LLM Support (Ollama)
- ☁️ API-based LLM Support (Mistral)

---

## 🛠️ Tech Stack

- Python
- Streamlit
- LangChain
- Ollama
- Mistral AI
- Tavily Search API
- BeautifulSoup
- Requests
- HTML Parsing

## ⚙️ Installation

### 1. Clone the Repository

```bash
git clone https://github.com/rahulmishra54/multi_agent_system.git
```

### 2. Navigate to the Project

```bash
cd multi_agent_system
```

### 3. Create Virtual Environment

```bash
python -m venv venv
```

### 4. Activate Virtual Environment

Windows

```bash
venv\Scripts\activate
```

Linux / macOS

```bash
source venv/bin/activate
```

### 5. Install Dependencies

```bash
pip install -r requirements.txt
```

---

## ▶️ Run the Application

```bash
streamlit run app.py
```
